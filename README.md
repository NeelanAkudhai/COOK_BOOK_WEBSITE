# COOK_BOOK_WEBSITE

Cookbook Website
The Cookbook website is an interactive platform where users can explore various recipes, express their opinions by liking or disliking recipes, and contribute their own recipes using a unique user ID. The platform also allows users to register comments on recipes and provides an admin interface for managing the site. Complete CRUD (Create, Read, Update, Delete) operations are supported for recipes and user data.
Key Features
• Recipe Browsing: View and explore a variety of recipes.
• User Interaction: Like or dislike recipes and register comments.
• User Contribution: Add your own recipes with a personalized user ID.
• Admin Interface: Manage recipes, comments, and user data through a dedicated admin page.
• CRUD Operations: Perform create, read, update, and delete operations seamlessly.
Technical Stack
• Backend Framework: Django
• Frontend Technologies: HTML, CSS
• Database: MongoDB
